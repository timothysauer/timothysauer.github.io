Additional Examples 1.1

1 Apply two steps of the Bisection Method on the interval \([1,2]\) to find the approximate root of \(f(x) = 2x^3-x-7.\)
Start with \(f(x)=2x^3-x-7\) on \([1,2]\), where \(f(1)<0\) and \(f(2)>0.\) The first step is to evaluate \(f(\frac{3}{2})=-\frac{7}{4}<0,\) which implies the new interval is \([\frac{3}{2},2].\) The second step is to evaluate \(f(\frac{7}{4})=\frac{63}{32}>0,\) giving the interval \([\frac{3}{2},\frac{7}{4}].\) The best estimate is the midpoint \(x_c=\frac{13}{8}.\)
2 Use the bisect.py code to find the solution of \(e^x = 3\) correct to six decimal places.
The commands 
def f(x): return np.exp(x)-3;
x=bisect(f,1,2,0.5e-6)
result in the approximate root \(1.098612,\) which agrees with \(\ln 3\) to six decimal places.
