Additional Examples 1.3

1 Find the multiplicity of the root \(r=0\) of \(f(x) = 6x-6\sin x-x^3.\)
The first five derivatives of \(f(x)\) evaluated at the root are: \begin{eqnarray*} f(x) &=& 6x-6\sin x -x^3 \longrightarrow f(0) = 0\\ f'(x) &=& 6-6\cos x-3x^2 \longrightarrow f'(0) = 6-6 = 0\\ f''(x) &=& 6\sin x - 6x \longrightarrow f''(0) = 0\\ f'''(x) &=& 6\cos x - 6 \longrightarrow f'''(0) = 6-6 = 0\\ f^{(4)}(x) &=& -6\sin x \longrightarrow f^{(4)}(0) = 0\\ f^{(5)}(x) &=& -6\cos x \longrightarrow f^{(5)}(0) = -6 \neq 0, \end{eqnarray*} showing that the multiplicity of the root \(r=0\) is 5.
2 Use the scipy command brentq with initial interval \( [-0.001,0.001] \) to approximate the root of \(f(x) = 6x-6\sin x-x^3\). Compute the forward and backward errors of the approximate root.
The scipy commands 
f=lambda x: 6*x-6*sin(x)-x**3
brentq(f,-1.,2.)
returns the approximate root \(x_a=-0.00015607658631472518\) instead of the exact root \(0.\) Therefore the forward error is \(|0-(-0.00015607658631472518)|=0.00015607658631472518.\) The backward error, found by computing \(f(x_a),\) is \( 2.013981227587752\times 10^{-20}\).
