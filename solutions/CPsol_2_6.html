\(\def\ds{\displaystyle} \) 1 (a) The Conjugate Gradient loop written in pseudocode in the textbook can be coded as follows. 
function x=cg(a,b,n)
% Inputs: symm. pos. def. matrix a, right-hand side b, number of steps n
% Output: solution x
x=zeros(length(b),1);
r=b-a*x;
d=r;d1(:,1)=d;r1(:,1)=r;
for i=1:n
    if max(abs(r))< eps break; end
    alf=d'*r/(d'*a*d);
    x=x+alf*d;
    rold=r;
    r=rold-alf*a*d;
    beta=r'*r/(rold'*rold);
    d=r+beta*d;d1(:,i+1)=d;r1(:,i+1)=r;
end
The test for \(r\) equal to zero uses eps, the machine epsilon. The Matlab command 
>> x=cg([1 0;0 2],[2;4],2)
returns the solution \(x=[2, 2]\).
