15 Replace the trapstep call in Computer Problem 6.3.11 with the folwing RK4 code: 
def rk4step(ydot,t, w, h):
    s1 = ydot(t, w)
    s2 = ydot(t + h/2, w + h*s1/2)
    s3 = ydot(t + h/2, w + h*s2/2)
    s4 = ydot(t + h, w + h*s3)
    return w + h*(s1 + 2*s2 + 2*s3 + s4)/6
Compared with the trapstep version, the oribts stay closed for a much longer time period.
