3 According to the Chebyshev formulas, the interpolation error of the degree \(n-1\) interpolation polynomial on the fundamental domain \([1,e]\) is at most \[\frac{\left(\frac{e-1}{2}\right)^n|f^{(n)}(c)|}{2^{n-1}n!} \] where \(|f^{(n)}(x)|=(n-1)!|x^{-n}|\) and \(1\leq c\leq e\). The maximum value of \(|f^{(n)}(c)|\) for \(1\leq c\leq e\) is \((n-1)!\), so the interpolation error is bounded above by \((e-1)^n/2^{2n-1}n\). Checking shows that this expression is less than \(0.5\times 10^{-10}\) for \(n=26\). Program 3.4 can be adapted to find the degree 25 interpolating polynomial at 26 Chebyshev nodes in \([1,e].\)

The part of Program 3.4 that moves any input between \(10^{-4}\) and \(10^4\) into the fundamental domain must also be changed. Since \(e^{10}>10^4\), the exponent \(k\) satisfying \(e^k\leq x\lt e^{k+1}\) can be found using Python code 

k = 10
while exp(k) > x:
    k = k-1
Now that \(xe^{-k}\) is in \([1,e]\), it can be substituted into the interpolation polynomial, and the result is added to \(k\) to get the approximate natural logarithm.
