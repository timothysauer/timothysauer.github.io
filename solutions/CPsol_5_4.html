9 Assuming you have written a python program adapsimp.py, similar to adapquad.py, but using Simpson's Rule, the myerf function can be written in two lines: 
def myerf(x):
    return 2*adapsimp(np.exp(-x^2),0,x,0.5e-8)/np.sqrt(pi)
Then myerf(1) returns \(0.84270079\) and myerf(3) returns \(0.99997791\) rounded to 8 decimal places, in agreement with the scipy ``error function'' scipy.special.erf.
