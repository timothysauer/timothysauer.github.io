5(a) \(f'(x) = 2x - 2\sin x, f'(0) = 0\)

\(f''(x) = 2 - 2\cos x, f''(0) = 0\)

\(f'''(x) = 2\sin x, f'''(0) = 0\)

\(f''''(x) = 2\cos x, f''''(0) = 2\)

Therefore multiplicity of \(f\) at the root \(r = 0\) is 4.

5(b) \(f'(x) = 4e^{2x}-8x - 4, f'(0) = 0\)

\(f''(x) = 8e^{2x} - 8, f''(0) = 0\)

\(f'''(x) = 16e^{2x}, f'''(0) = 16\)

Therefore multiplicity of \(f\) at the root \(r = 0\) is 3.

5(c) \(f'(x) = \cos x + 1, f'(\pi) = 0\)

\(f''(x) = -\sin x, f''(\pi) = 0\)

\(f'''(x) = -\cos x, f'''(\pi) = 1\)

Therefore multiplicity of \(f\) at the root \(r = 0\) is 3.

5(d) \(f'(x) = 18\cos 3x + 81x^2-18, f'(0) = 0\)

\(f''(x) = -54\sin 3x + 162x, f''(0) = 0\)

\(f'''(x) = -162\cos 3x + 162, f'''(0) = 0\)

\(f^{(4)}(x) = 486\sin 3x, f^{(4)}(0) = 0\)

\(f^{(5)}(x) = 1458\cos 3x, f^{(5)}(0) = 1458\)

Therefore multiplicity of \(f\) at the root \(r = 0\) is 5.

5(e) \(f'(x) = -6\cos x\sin x+ 6x -4x^3 = -3\sin 2x+6x-4x^3, f'(0) = 0\)

\(f''(x) = -6\cos(2x + 6 - 12x^2), f''(0) = 0\)

\(f'''(x) = 12\sin 2x -24x, f'''(0) = 0\)

\(f^{(4)}(x) = 24\cos 2x - 24, f^{(4)}(0) = 0\)

\(f^{(5)}(x) = -48\sin 2x, f^{(5)}(0) = 0\)

\(f^{(6)}(x) = -96\cos 2x, f^{(6)}(0) = -96\)

Therefore multiplicity of \(f\) at the root \(r = 0\) is 6.

7 \(f'(x) = 2\cos 2x - 2 - 3cx^2, f'(0) = 0\)

\(f''(x) = -4\sin 2x - 6cx, f''(0) = 0\)
