COMPUTER PROBLEMS 11.4

\(\def\ds{\displaystyle} \) 1 A Matlab program that carries out the computations of Example 11.9 is the following: 
function xout=mdctexample(x,n)
len=length(x);   % input column vector x
for i=1:n
  for j=1:2*n
    M(i,j)=sqrt(2/n)*cos((i-0.5)*(j-0.5+n/2)*pi/n);
  end
end
N=M';
xout=[];
for k= 1:len/n-2
  v1=M*x((k-1)*n+1:(k+1)*n);
  v2=M*x(k*n+1:(k+2)*n);
  wa=N*v1;
  wb=N*v2;
  w2=wa(n+1:2*n);
  w3=wb(1:n);
  u2=(w2+w3)/2;
  xout=[xout;u2];
end
Applying the code to part (a), for example, would use the Matlab command 
>> x=(1:12)';xout=mdctexample(x,4)

xout =

    5.0000
    6.0000
