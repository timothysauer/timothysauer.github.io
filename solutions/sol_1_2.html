If \(x_0\ge r\), then \(x_1-r = g'(c)(x_0-r)\lt x_0-r\), and \(x_1-r = g'(c)(x_0-r)\ge 0\), so \(r \le x_1 \lt x_0\). If \(x_0\le r\), then \(r-x_1 = g'(c)(r-x_0)\lt r-x_0\), and \(r-x_1 = g'(c)(r-x_0)\ge 0\), so \(x_0 \lt x_1 \le r\). In either case, since \(r\) and \(x_0\) belong to \([a,b]\), \(x_1=g(x_0)\) does also, and by extension, so does \(x_2\), \(x_3\), etc. Finally, \(|x_1-r|\leq B|x_0-r|\) extends to \(|x_i-r|\leq B^i|x_0-r|\), which converges to zero as \(i\to\infty\).
