\(\def\ds{\displaystyle} \) 1 (a) Begin by applying the DCT to \([3,3]^T\): \[y=Cx= \left[\begin{array}{cc} \frac{1}{\sqrt{2}}&\frac{1}{\sqrt{2}}\\ \cos \frac{\pi}{4}&\cos \frac{3\pi}{4} \end{array}\right] \left[\begin{array}{c} 3\\3\\ \end{array}\right]= \left[\begin{array}{rr} \frac{1}{\sqrt{2}}&\frac{1}{\sqrt{2}}\\ \frac{1}{\sqrt{2}}&-\frac{1}{\sqrt{2}}\\ \end{array}\right] \left[\begin{array}{c} 3\\3\\ \end{array}\right]= \left[\begin{array}{c} 3\sqrt{2}\\0\\ \end{array}\right] \] Theorem 11.2 provides the interpolating function \[P_2(t)=\frac{1}{\sqrt{2}}y_0+y_1\cos\frac{(2t+1)\pi}{4}\] for the data \((0,3), (1,3)\). Since \(y_0=3\sqrt{2}\) and \(y_1=0\), this reduces to \(P_2(t)=3\).

1 (b) The DCT is \[y= \left[\begin{array}{rr} \frac{1}{\sqrt{2}}&\frac{1}{\sqrt{2}}\\ \frac{1}{\sqrt{2}}&-\frac{1}{\sqrt{2}}\\ \end{array}\right] \left[\begin{array}{r} 2\\-2\\ \end{array}\right]= \left[\begin{array}{c} 0\\2\sqrt{2}\\ \end{array}\right] \] The interpolating function is \begin{eqnarray*} P_2(t)&=&\frac{1}{\sqrt{2}}y_0+y_1\cos\frac{(2t+1)\pi}{4}\\ &=&2\sqrt{2}\cos\frac{(2t+1)\pi}{4} \end{eqnarray*} 1 (c) The DCT is \[y= \left[\begin{array}{rr} \frac{1}{\sqrt{2}}&\frac{1}{\sqrt{2}}\\ \frac{1}{\sqrt{2}}&-\frac{1}{\sqrt{2}}\\ \end{array}\right] \left[\begin{array}{r} 3\\1\\ \end{array}\right]= \left[\begin{array}{r} 2\sqrt{2}\\\sqrt{2} \end{array}\right] \] The interpolating function is \begin{eqnarray*} P_2(t)&=&\frac{1}{\sqrt{2}}y_0+y_1\cos\frac{(2t+1)\pi}{4}\\ &=&2+\sqrt{2}\cos\frac{(2t+1)\pi}{4}. \end{eqnarray*} 1 (d) The DCT is \[y= \left[\begin{array}{rr} \frac{1}{\sqrt{2}}&\frac{1}{\sqrt{2}}\\ \frac{1}{\sqrt{2}}&-\frac{1}{\sqrt{2}}\\ \end{array}\right] \left[\begin{array}{r} 4\\-1\\ \end{array}\right]= \left[\begin{array}{r} \frac{3}{2}\sqrt{2}\\\frac{5}{2}\sqrt{2} \end{array}\right] \] The interpolating function is \begin{eqnarray*} P_2(t)&=&\frac{1}{\sqrt{2}}y_0+y_1\cos\frac{(2t+1)\pi}{4}\\ &=&\frac{3}{2}+\frac{5\sqrt{2}}{2}\cos\frac{(2t+1)\pi}{4} \end{eqnarray*}

5 The half-angle formula for cosine is \[\cos\frac{\theta}{2}=\sqrt{\frac{1+\cos\theta}{2}}.\] Therefore \begin{eqnarray*} b&=&\frac{1}{\sqrt{2}}\cos \frac{\pi}{8} = \sqrt{\frac{1+\cos\frac{\pi}{4}}{4}} =\frac{\sqrt{2+\sqrt{2}}}{2\sqrt{2}}\\ c&=&\frac{1}{\sqrt{2}}\cos \frac{3\pi}{8} = \sqrt{\frac{1+\cos\frac{3\pi}{4}}{4}} =\frac{\sqrt{2-\sqrt{2}}}{2\sqrt{2}}\\ \end{eqnarray*} Using the identities \(\cos(2\pi-\theta)=\cos\theta\) and \(\cos(\pi-\theta)=-\cos \theta\), the rest of the matrix entries can be identified with \(\pm a,b\) or \(c\).
